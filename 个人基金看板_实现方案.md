## 四、推荐方案 B：Python + 本地 HTML 看板

### 架构

```
┌──────────────┐     ┌──────────────────┐     ┌──────────────┐
│  天天基金API   │────▶│  Python 数据引擎   │────▶│  本地HTML看板  │
│  东方财富API   │     │  (requests拉数据)  │     │  (浏览器打开)  │
└──────────────┘     └────────┬─────────┘     └──────┬───────┘
                              │                      │
                              ▼                      ▼
                     ┌──────────────┐     ┌──────────────┐
                     │  fund_data   │     │  基金净值走势   │
                     │  .json       │     │  实时估值看板   │
                     │  (本地缓存)   │     │  持仓盈亏计算   │
                     └──────────────┘     └──────────────┘
```

### 核心功能（MVP）

1. **持仓总览**：你持有的基金列表，实时估值、今日涨跌、持仓盈亏
2. **净值走势图**：近30天/近1年净值曲线（ECharts）
3. **资产配置饼图**：按基金类型的仓位分布
4. **手动维护**：一个 `my_funds.json` 配置文件记录你的基金代码和持仓

### 技术栈

| 层 | 技术 | 原因 |
|----|------|------|
| 数据抓取 | Python `requests` + `re` | 简单直接 |
| 数据存储 | JSON文件（本地） | 零依赖，个人够用 |
| 前端展示 | 纯HTML + ECharts CDN | 不需要框架，一个文件搞定 |
| 图表 | ECharts | 折线图、饼图、K线样式齐全 |
| 自动刷新 | `setInterval` 每30秒拉一次估值 | 模拟实时看盘 |
